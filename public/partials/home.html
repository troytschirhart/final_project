Click on the 'Register' button to register.

If you're already registered, click on the 'Login' button to login.

This app can be used to find the total value of a portfolio of stocks. Once a user logs in, they can enter the stock symbols and number of shares of each stock, and the app will pull the latest stock prices. It will then calculate the current value of each stock held, and will sum the result to display the total value of the portfolio.
